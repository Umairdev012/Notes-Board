# 📝 Notes Board App

A simple and modern **Notes App** built with **HTML, CSS, and Vanilla JavaScript**.  
It allows users to create, edit, delete, and store notes using **localStorage**.

---

## ✨ Features

- 📝 Add new notes instantly
- ✏️ Edit notes anytime
- 🗑️ Delete notes easily
- 💾 Saves data in browser (localStorage)
- 📱 Fully responsive design
- 🎨 Clean modern UI
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
